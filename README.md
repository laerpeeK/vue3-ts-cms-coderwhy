## 此项目说明

### 技术栈
**Vue3 + Vue-Router + Vuex + TypeScript + ElementPlus + scss + Axios**

### 项目功能
此项目为后台管理系统学习
支持
 + 商品数据的可视化
 + 用户，菜单，角色的数据展示
 + 用户，角色的增删改查
